Trabalhando com ID e Classe

Setores ID

É usado um seletor de id em um element HTML para deixar ele único e específico.

deve ser usado uma única vez na página.

Exemplo

USANDO UM ID

Seletores de Classe

O seletor de classe seleciona elemento com um atributo de classe especifico.

Pode ser usado mais de uma vez na página

Exemplo:

USANDO CLASSE

USANDO CLASSE

USANDO CLASSE

OUTRO TEXTO
